# Комплексная настройка корпоративной сети: сегментация, автоматизация адресации, авторизации и выход в интернет <br>
Настроить безопасную, сегментированную сеть с автоматической раздачей IP-адресов, возможностью управления устройствами и доступом в интернет.<br>
                       

Сеть между маршрутизаторами (WAN): 100.100.100.0/30<br>
R1 G0/1: 100.100.100.1<br>
R2 G0/1: 100.100.100.2<br>

Внутренняя сеть R1 (LAN):<br>
G0/0.4 к серверам 192.168.4.1/24<br>
G0/0.2 к пользователям 192.168.2.1/24<br>
G0/0.3 к пользователям 192.168.3.1/24<br>
G0/0.5 к коммутаторам 192.168.5.1/24<br>

Задачи:<br>
VLAN: Создать и настроить VLAN 2, 3, 4, 5 на коммутаторах .<br>
DHCP: Настроить  DHCP-сервера для VLAN 2, 3. <br>
NAT: Настроить на R1 трансляцию адресов (PAT) для выхода в "интернет" (к R2 и PC1).<br>
SSH: Настроить удаленное управление по SSH на router1 , core-Switch , Switch2 , Switch1 .<br>
RADIUS: настроить авторизацию через RADIUS-server на  router1 , core-Switch , Switch2 , Switch1 .<br>
Маршрутизация: Настроить статическую маршрутизацию между сетями.<br>
